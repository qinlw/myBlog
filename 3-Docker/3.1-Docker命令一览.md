# Docker 常用命令总结

## 容器管理

- 运行容器（前台模式）

	```docker run <image_name>```

- 运行容器（后台模式）

	```
	docker run -d <image_name>
	docker run --name <container_name> -p <host_port>:<container_port> <image_name>  // 指定容器名称和端口映射
	```

- 查看运行中的容器

	```
	docker ps
	docker ps -a // 查看所有容器（包括已停止的）
	```

- 启动已停止的容器

	```docker start <container_name/ID>```

- 停止运行中的容器

	```docker stop <container_name/ID>```

- 强制停止容器

	```docker kill <container_name/ID>```

- 重启容器

	```docker restart <container_name/ID>```

- 查看容器状态

	```docker stats```

- 进入运行中的容器终端

	```docker exec -it <container_name/ID> /bin/bash```

- 查看容器日志

	```docker logs <container_name/ID>```

- 删除已停止的容器

	```	docker rm <container_name/ID>	docker rm -f <container_name/ID> // 强制删除运行中的容器	```## 镜像管理- 检索镜像	```docker search```- 拉取镜像

	```docker pull <image_name>:<tag>```

- 查看本地镜像

	```docker images```

- 删除镜像

	```docker rmi <image_name/ID>```

- 认证 (必须登录后才能推送镜像到仓库)

	```docker login```

- 提交修改

	```docker commit```

- 命名，标记镜像

	```docker tag```

- 推送镜像到仓库

	```docker push <image_name>:<tag>```

- 保存，镜像导出为离线文件

	```docker save```

- 加载，从离线文件加载镜像

	```docker load```

- 清理未使用的镜像

	```docker image prune```

- 构建镜像（需在Dockerfile目录执行）

	```docker build -t <image_name>:<tag> .```

## 网络管理

- 创建自定义网络

	```docker network create <network_name>```

- 查看网络列表

	```docker network ls```

- 将容器连接到网络

	```docker network connect <network_name> <container_name>```

## 数据卷管理

- 创建数据卷

	```docker volume create <volume_name>```

- 查看数据卷

	```docker volume ls```

- 删除未使用的数据卷

	```docker volume prune```

## DockerCompose

- 启动服务（后台模式）

	```docker-compose up -d```

- 停止并删除服务

	```docker-compose down```

- 查看服务状态

	```docker-compose ps```

- 查看服务日志

	```docker-compose logs```

## DockerSwarm

- 初始化Swarm集群

	```docker swarm init```

- 加入工作节点

	```docker swarm join --token <token> <manager-ip>:2377```

- 部署服务

	```docker service create --name <service_name> <image_name>```

- 查看服务列表

	```docker service ls```

- 扩展服务实例数量

	```docker service scale <service_name>=<replicas>```

## 实用命令

- 查看Docker系统信息

	```docker info```

- 查看容器资源使用情况

	```docker stats```

- 查看容器进程

	```docker top <container_name/ID>```

- 复制文件到容器

	```docker cp <local_path> <container_name/ID>:<container_path>```
